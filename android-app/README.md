# MusicApp Android项目

## 项目结构

```
app/
├── src/main/java/com/musicapp/
│   ├── MainActivity.kt
│   ├── App.kt
│   ├── api/
│   ├── model/
│   ├── ui/
│   ├── repository/
│   ├── util/
│   └── service/
├── src/main/res/
└── build.gradle.kts
```

## 构建说明

使用Android Studio打开此项目，或使用命令行构建：
```bash
./gradlew assembleDebug
```

## 最低要求

- Android 7.0 (API 24)
- Kotlin 1.9+
